1085. Perfect Sequence (25)

Given a sequence of positive integers and another positive integer p. The sequence is said to be a "perfect sequence" if M <= m * p where M and m are the maximum and minimum numbers in the sequence, respectively.

Now given a sequence and a parameter p, you are supposed to find from the sequence as many numbers as possible to form a perfect subsequence.

Input Specification:

Each input file contains one test case. For each case, the first line contains two positive integers N and p, where N (<= 105) is the number of integers in the sequence, and p (<= 109) is the parameter. In the second line there are N positive integers, each is no greater than 109.

Output Specification:

For each test case, print in one line the maximum number of integers that can be chosen to form a perfect subsequence.

Sample Input:

10 8 2 3 20 4 5 1 6 7 8 9

Sample Output:

8

Solution:

思路： 这个题肯定是不能用用O(n^2)的算法的。那么如何优化呢，其实一般来说我们会先考虑优化到nlog(n)，然后再进一步 到O(n)。 这道题的复杂度可以到nlog(n):排序和二分。

  • 首先对数组进行排序
  • 从头往后（小-->大），对于每一个m（a[i]）,得到mp，然后在[i,n-1]二分搜索mp，因为M<mp,那么只需要找到mp 应该插入的位置即可，（不过要注意，二分返回的位置是0--n，可以是刚好找到，也可以是没有找到但是给出了 应该插入的位置，这两种情况要分开考虑）
  • 注意m*p会溢出，用unsigned long long

代码： 

#include <iostream>
#include <algorithm>
#include <string.h>

using namespace std;

int b_search(int *a, int l, int r, unsigned long long target) {
    while(l<=r) {
        int mid = l+((r-l)>>1);
        if(a[mid]>target) {
            r = mid-1;
        } else if(a[mid]<target) {
            l = mid+1;
        } else {
            return mid;
        }
    }
    return l;
}

int main()
{
    //freopen("input.txt", "r", stdin);
    int n, p;
    cin>>n>>p;
    int a[100000+1];
    memset(a, 0, sizeof(a));
    for(int i=0; i<n; i++) {
        cin>>a[i];
    }
    sort(a,a+n);
    int ret = 0;
    for(int i=0; i<n; i++) {
        unsigned long long mp = a[i]*p;
        int M = b_search(a,i,n-1,mp);
        if(M==n) {
            if(ret < M-i)
                ret = M-i;
        }
        else if(a[M] != mp) {
            if(ret < M-i)
                ret = M-i;
        }
        else {
            if(ret <M-i+1)
                ret = M-i+1;
        }


    }
    cout<<ret;
    return 0;
}
